Types :

1.Logical And (&&)
2.Logical Or 
3.Not

-Logical And (&&):

Returns True if Both operands are true.

console.log(true && true);
> true

console.log(true && false);
> false


-Logical Or (||):

True if one of the operands is true.

console.log(true || true);
> true

console.log(true || false);
> true

console.log(false || false);
> false


-Not(!):

Returns opposite of passed operands.

console.log(!true);
> false

console.log(!false)
> true
